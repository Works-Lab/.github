# Works Lab Brand

This folder holds placeholder visual identity files for Works Lab.

`works-lab-mark.svg` is a simple temporary mark that can be used while the public GitHub organization is being set up. It is intentionally minimal and should not be treated as a final identity system.

If GitHub does not accept the SVG directly for the organization avatar, export it to PNG first and upload the PNG in organization settings.

Future visuals should stay minimal, practical, and easy to reproduce across README files, photos, diagrams, and small interface surfaces.
