# Works Lab Setup Checklist

Manual setup steps:

- Upload the Works Lab avatar in GitHub organization settings.
- Confirm the public profile README is visible at `https://github.com/Works-Lab`.
- Create or confirm the `oto` and `trace` repositories are public.
- Pin `oto` and `trace` on the organization profile.
- Invite Bryan to the organization.
- Add or confirm repository descriptions and topics.
- Decide licenses before publishing reusable CAD, hardware, firmware, or software files.
- Add real prototype photos or renders when they are ready.
